# **App Name**: Markup Maestro

## Core Features:

- HTML Preview: Real-time HTML Preview: Dynamically render the HTML code as it is being written.
- Tag Autocomplete: Basic Tag Autocompletion: Suggest common HTML tags as the user types, speeding up the coding process.
- AI HTML Generator: Generate a simple HTML page based on user prompts using a tool, offering a starting point for development.
- Downloadable HTML: Downloadable HTML File: Allow users to download the generated HTML as a .html file.
- Code Formatting: Code Formatting: Automatically format HTML code for better readability and consistency.

## Style Guidelines:

- Primary color: Vivid purple (#9F5BBA) to evoke creativity and coding.
- Background color: Dark gray (#333333) for a professional, coding-friendly environment.
- Accent color: Electric green (#39FF14) for highlights and calls to action.
- Body and headline font: 'Inter', a sans-serif font, providing a modern and clean look for both headings and body text.
- Code Font: 'Source Code Pro' to clearly render all HTML elements being rendered.
- Use minimalist, geometric icons for functions like download, format, and AI generation.
- Split-screen layout with the HTML code editor on one side and the real-time preview on the other.
